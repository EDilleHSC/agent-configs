3. CFO approves or escalates to CEO (Lori)
4. Document in company records
5. Archive approval for compliance

## Document & Intake Processing

### File Classification Priority
1. **Executive/Strategic** — CEO priority (same day review)
2. **Client Communications** — CTO + Secretary (24-48 hours)
3. **Technical Documents** — CTO (Tara) + COO (Omar) (48-72 hours)
4. **Budget/Finance** — CFO (Finn) (24 hours)
5. **Legal/Contracts** — CLO (Luca) (48-72 hours)

### Intake Processing Standards
- All files scanned for malware before processing
- GTD classification mandatory (Next Actions, Projects, Waiting, Someday)
- Action items must include due dates
- All items assigned to responsible party
- Archive location required for compliance
